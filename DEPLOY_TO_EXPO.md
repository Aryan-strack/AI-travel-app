# 🚀 Deploy Smart Travel Planner to Expo Development Build

## 📱 **Step-by-Step Instructions:**

### **Step 1: Create EAS Project**
1. **Run this command manually:**
   ```bash
   npx eas init
   ```
2. **When prompted, type:** `Y` and press Enter
3. **This will create a project in your Expo account**

### **Step 2: Deploy to Expo**
1. **After project creation, run:**
   ```bash
   npx eas update --branch main --message "Deploy Smart Travel Planner"
   ```
2. **This will deploy your app to Expo's servers**

### **Step 3: Share Your App**
1. **Your app will be available at:**
   ```
   https://expo.dev/@hassan1122q/smart-travel-planner-hassan
   ```
2. **Share this link with your friend**
3. **They can install "Expo Go" app and scan the QR code**
